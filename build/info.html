About this application

HWiNFO Summarize v0.1.0

Google Chrome application.

Summarize the HTM output files from the HWiNFO application to display at a glance the most relevant information about your PC.

Save a summary file in standard HTML.

Author:

Daniel Sosa: www.danielsosa.uy/dev

Dependencies:

  • Cheerio
  • Roboto

  • Original icon
  • Batch open and save functionality
